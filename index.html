Projects Maven Plugin

Introduction

This is a Maven plugin to perform operations on all or a subset of projects in a multi-module build. It can be used to list projects, add dynamically during build dependencies to a (sub)project, or create a BOM (Bill Of Materials) project. It supports includes and excludes patterns to filter projects.
